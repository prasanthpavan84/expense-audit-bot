# Contributing to Expense Audit Bot

We welcome contributions from the community! Please read through these guidelines to get started.

## How to Contribute
1. **Report Bugs**: Open an issue describing the bug, steps to reproduce, and expected vs actual behavior.
2. **Submit Pull Requests**:
   - Create a feature branch from `develop`.
   - Ensure all code modifications adhere to PEP 8 standards.
   - Run the full test suite (`uv run pytest tests/`) before submitting.
   - Ensure your code coverage remains high.
   - Update `CHANGELOG.md` with your additions.
